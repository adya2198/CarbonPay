# CarbonPay

## Description
CarbonPay is designed to facilitate payments and transactions while being environmentally friendly. The platform aims to reduce carbon emissions associated with traditional payment methods.

## Features
- Secure transactions
- User-friendly interface
- Environmentally conscious

## Installation
To install CarbonPay, follow these steps:
1. Clone the repository: `git clone https://github.com/adya2198/CarbonPay.git`
2. Navigate to the project directory.
3. Install dependencies: `npm install`

## Usage
1. Start the server: `npm run dev`
2. Open your browser and go to `http://localhost:5173`
